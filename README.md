# E-commerce

Implementación de una tienda virtual, en base al MVC, usando de api's como mercado libre, paypal y también usando librería como minicart para el carro de compras.

## Herramientas
+ Jquery
+ Ajax
+ Mercado libre api
+ Bootstrap 4
+ Html 5
+ Css
